# 金融市场部完成"民生FICC配置指数"方法论研发

金融市场部近期完成"民生FICC配置指数"（CMBC FICC Allocation Index）方法论设计与历史回测验证。该指数采用风险预算驱动的系统化配置方法，覆盖中国国债、G7货币与大宗商品三大类资产，以等风险贡献方法确定配置权重，并叠加Carry、动量、价值三类经典因子形成基准与增强双轨体系，填补了我行交易账簿在跨FICC资产业绩比较基准方面的空白。最长近10年的历史回测显示，5年期窗口下基准指数年化收益3.86%、夏普比率1.03，增强指数在回撤基本不变的前提下将年化收益提升至4.17%、夏普比率提升至1.13。目前已交付完整方法论说明书、交互式回测分析工具及标准化月度报告模板。需要说明的是，该方法论本身采用业界成熟、透明、可审计的经典量化框架，不涉及黑箱模型；而整个研发过程——从方法论设计、回测系统搭建到文档撰写——全程由大模型深度参与协同完成，是部门"AI辅助量化研究"提升效率的一次实践。

该指数面向多层用户提供参考：对交易账簿，提供方法论严谨、规则透明的跨资产业绩比较基准，替代以往拼凑式复合基准；对金融市场部资产配置，风险预算分配结果可作为国债、外汇、商品间战略配置比例的量化参考；对各交易团队，指数仅提供系统化Beta层面参考，团队可在此基准之上结合自身判断与能力叠加增厚Alpha，二者互补而非替代；长远看，国内市场尚缺乏被广泛引用的跨FICC风险平价基准，该指数体系亦具备未来对外发布、成为行业参考标准或产品挂钩标的的潜力。

下一步，将把样本空间扩展至完整目标资产范围（白银、原油、铜及其余G7货币），建立常态化月度再平衡与业绩归因机制，邀请部分交易团队试用并收集反馈，待运行机制稳定后再评估对外发布的时点与口径。

*本指数历史模拟表现基于规则化回溯计算，不代表未来表现，不构成投资建议。完整说明书与回测数据请联系金融市场部获取。*
